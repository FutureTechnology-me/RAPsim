Detailed Description

Provides most methods for editing the GridModel.

Initializes the GridMapListener. Loads images for the JGM (loadImagesNew) Adds and deletes GridObjects. Provides logics for neighbor detection and power-line drawing.

Author
Kristofer Schweiger, Sabrina Huber
